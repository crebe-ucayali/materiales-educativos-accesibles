# Bitácora de cambios

Este documento registra los cambios principales realizados en el repositorio Materiales Educativos Accesibles.

La bitácora permite mantener un historial ordenado del crecimiento del proyecto, las mejoras realizadas, los materiales incorporados y los ajustes de accesibilidad o estructura.

## Versión inicial

### 2026

- Creación del repositorio Materiales Educativos Accesibles.
- Definición del proyecto como recurso educativo digital personal.
- Redacción del README con enfoque personal, educativo y no comercial.
- Creación de documentos base en la carpeta `docs`.
- Organización inicial de criterios de autoría, fuentes, uso permitido, alcance pedagógico, estructura y adaptación de materiales.

## Cambios por registrar

Se recomienda registrar cada cambio importante con los siguientes datos:

- Fecha del cambio.
- Archivo o sección modificada.
- Tipo de cambio realizado.
- Motivo del cambio.
- Observaciones pendientes.

## Tipos de cambios sugeridos

- Incorporación de nuevos materiales.
- Ajustes de accesibilidad visual.
- Corrección de textos o instrucciones.
- Reorganización de carpetas.
- Actualización de fuentes y créditos.
- Mejora de fichas descargables.
- Creación de nuevas secciones.
- Revisión pedagógica de materiales.

## Formato sugerido

```text
Fecha:
Archivo o sección:
Cambio realizado:
Motivo:
Observaciones:
```

## Nota

La bitácora no reemplaza el historial de commits de GitHub. Su función es explicar de manera sencilla los cambios más relevantes para fines educativos, organizativos y de revisión del proyecto.
